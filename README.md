# RNN-HeartAttack-Prediction

Installation Prerequisites:


```bash
brew cask install anaconda
conda create --name rnn_heartattack_env python=3.5
conda info --envs
source activate rnn_heartattack_env
conda list
pip install -r requirements.txt
python rnn.py
```
